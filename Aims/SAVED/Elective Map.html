2.11.6 ELECTIVE MAP

          This form is used to map electives to semesters.

Ø  Course*: Select a course from list.

Ø  Batch*: Select a batch from list.

Ø  Semester*: Select a semester from list.

Ø  Elective*: Select an elective subject from list.

 

NOTE:Generated grid view contains Student Code, Student name and subject. Select subject from list for each student and click update.
