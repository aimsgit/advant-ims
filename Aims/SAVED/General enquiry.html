2.7    ADMISSION MODULE

2.7.1 GENERAL ENQUIRY

                  

 

                   This form can be used to capture details of people who comes to enquire about admission, courses or other details of school/college.

 

         

Ø  Branch*: Select a Branch name.

Ø  Enquiry No.*: Enter enquiry number.

Ø  Aadhaar/ NIC No:  Enter Aadhar/NIC No.

Ø  Date*: Enter Date on which enquiry is done.

Ø  Title*: Select a Title.

Ø  Name*: Enter Name of the student.

Ø  Date of Birth: Enter date of birth of the student

Ø  Age: It is auto filled based on DOB.

Ø  Street address: Enter address where student is staying.

Ø  City: Enter city name.

Ø  Pin code: Enter pin code of that place.

Ø  District: Enter district name.

Ø  State*: select a state Name.

Ø  Country: enter country name.

Ø  Phone No.: Enter phone number.

Ø  Mobile No*: Enter mobile number.

Ø  Caste: Enter Caste of the student.

Ø  Mother’s Name: Enter parent’s name.

Ø  Counselor: Enter Counselor’s name.

Ø  Parent’s Occupation: Enter parent’s occupation.

Ø  Annual Income: Enter annual income of parent.

Ø  Email: Enter Email Id of parent.

Ø  Qualification: enter qualification of parent.

Ø  Enquiry related to*: Select a field from list.

Ø  Course Type*: select a course type from list.

Ø  Course name*: select a course name from list.

Ø  Source of Information*: select a source from drop down list.

Ø  Remarks: If any remarks are there fill it here.

Ø  Prospectus given: select any radio button – Yes/No.

 

 

 

NOTE: Added data can be viewed in grid view which includes enquiry no, Branch name, date , time, name, phone no, mobile no, Email, street address, city, pin code, district, state, country, counselor, caste, enquiry related to, course type, course name, source of information, prospectus, remarks, parent’s name, parent’s occupation, annual income. It also has edit and delete facility.
