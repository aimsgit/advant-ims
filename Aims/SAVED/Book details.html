2.17.2  BOOK DETAILS

 

            This form is used to enter Book details. Which can be used in library.

Ø  Book Name*: Enter name of the book received.

Ø  Book Code*: Enter Book code.

Ø  Receive date: Enter date on which book is received.

Ø  Author: Enter author of book.

Ø  Publisher: Enter publisher name.

Ø  Subject: Select subject to which book belongs.

Ø  Edition No: Enter edition number.

Ø  No of Pages: Enter number of pages.

Ø  Quantity*: By default quantity is 1.

Ø  Price: Enter price of the book.

 

NOTE:  Added data can be viewed in grid view which includes Book name, Book Code, receive date, Author, Book publisher, Subject name, Book edition Number, No. of Pages, Quantity, Price. It also has edit and delete facility.
