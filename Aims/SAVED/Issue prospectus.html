2.7.2 ISSUE PROSPECTUS

         Prospectus issued details can be captured here.      

Ø  Name*: Enter name to whom prospectus is given.

Ø  Prospectus no.*:  Enter prospectus number.

Ø  Entry Date*: Enter date on which prospectus is given.

Ø  Quantity*: Enter number of prospectus given.

Ø  Receipt No: Enter receipt number.

Ø  Price*: Enter price of prospectus given.

Ø   Remarks: This field is meant for any remarks which can be added about the prospectus.

 

 

NOTE:  Added data can be viewed in grid view which includes Name, Entry date, prospectus no, price, and Receipt no, Quantity, Remarks. It also has edit and delete facility.
