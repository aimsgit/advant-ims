2.18.12 BANK RECONCILIATION

 

          Here we can see Bank Reconciliation report details.

Ø  Bank: Select a bank from list.

Ø  From Date*: Enter date from where we want Bank Reconciliation report.

Ø  To Date*: Enter date till where we want to generate Bank Reconciliation report.

Ø  Select Table*: Select a field of which we want to see details in grid view.

 

NOTE:  After selecting bank click on report. A pop up window appears which shows Bank Reconciliation report as shown above.
