2.19.7 LOAN MASTER

      

          This form is used to enter Loan details taken by employees.

Ø  Employee Code*: Enter employee code.

Ø  Loan Number*: Enter loan number.

Ø  Loan Type: Enter loan type.

Ø   Loan Date*: Enter Loan date.

Ø   Loan Amount*: Enter loan amount.

Ø   Interest Rate: Enter interest rate.

Ø   Cheque No & Bank: Enter cheque number and bank name.

Ø   Cheque Date: Enter cheque date.

Ø  Monthly Deduction: Enter monthly deduction.

Ø  Balance loan: Enter balance loan.

Ø  Start date*: Enter start date of loan deduction.

 

NOTE: Added data can be viewed in grid view which includes Employee Name, Loan Number, Loan Type, Loan Date, Loan Amount, Interest Rate, Cheque No & Bank, Cheque Date, Monthly Deduction, Balance loan, start date. It also has edit and delete facility.
