2.8    STUDENTS

2.8.1 ENTER ATTENDANCE

 

 

This form can be used to take attendance of students and message can be sent to parents of absentees.

 

Ø  Academic Year*: Select academic year from the list.

Ø  Batch*: Select a batch from list to which attendance has to be generated.

Ø  Semester*: Select a semester from list.

Ø  Subject: Select a subject from the list.

Ø  Class Type: select a class type from the list.

Ø  Attendance Date*: select attendance date.

Ø  Period No.: Enter period number.

 

NOTE:  Added data can be viewed in grid view which includes Student Name, Student code,  Period No, Elective and check box for marking whether student is present or not. To put attendance click the check box of the student present and click on update.
