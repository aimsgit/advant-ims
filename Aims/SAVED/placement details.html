2.23.3 PLACEMENT/ TRAINING DETAILS

 

 

         This form can be used to enter placement details.

 

Ø  Student Code*: Enter code of the student.

Ø  Student Name: Enter name of the student.

Ø  Company Name*: Enter name of the company.

Ø  Salary: Enter the salary offered.

Ø  Placement Date: Enter placement date.

Ø  Description: Write description.

Ø  Remarks: This field is meant for any remarks which can be added about the Placement/Training details.

NOTE: Added Data can be viewed in the grid which includes Batch, Student Name, Student Code, Card Type, Issue Date with Edit and Delete facility.
