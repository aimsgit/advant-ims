2.20.1 PROJECT MASTER

 

This form is used to enter project details.

Ø  Project Name*: Enter project name.

Ø  Description: Enter description.

Ø  Submitted By*: Enter employee code and select who submitted project.

Ø   Submitted Date*: Enter submitted date.

Ø  Approved By: Enter employee code and select who approved project.

Ø  Approved Date: Enter approved date.

Ø  Project Start Date:  Enter project start date.

Ø  Project End Date: Enter project end date.

Ø  Donor/ Sponsor:  select a Donor/Sponsor name. This can be used while entering endowment project details.

 

NOTE: Added Data can be viewed in the grid which includes Project Name, Description, Submitted By, Submitted Date, Approved By, Approved Date, Project Start Date, and Project End Date. It also has edit and delete facility.
